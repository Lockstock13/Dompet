# dompet. 💸

Personal finance tracker dengan framework **Y = C + I + S**

## Features
- 💬 Chat input natural language (AI powered via Gemini)
- 📊 Dashboard dengan health score & visualisasi
- ✏️ Form input manual
- 📋 Riwayat transaksi + search & filter
- 📥 Export CSV (per bulan / semua)
- 🖨️ Laporan bulanan (Print / Save as PDF)
- 📱 Installable PWA (Add to Home Screen)
- ⚙️ Settings API key per user

## Setup

```bash
npm install
npm run dev
```

## Deploy ke Vercel
1. Push ke GitHub
2. Connect repo di vercel.com
3. Deploy otomatis!

## AI Parsing
Pakai Gemini API (ada free tier; limit mengikuti quota Google AI Studio).
Dapat API key di: https://aistudio.google.com
